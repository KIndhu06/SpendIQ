# 💰 SpendiQ – Smart Family Finance Management System

## 📌 Project Description

**SpendiQ** is an intelligent web-based finance management system designed to help families manage their daily financial activities.
The application allows users to track income, monitor expenses, analyze spending patterns, and predict future expenses using machine learning techniques.

The system provides a simple interface where users can upload bank statements, manage bills, set financial goals, and generate financial reports.

---

## 🎯 Objectives

* Help families manage income and expenses effectively
* Analyze spending behavior
* Predict future expenses using machine learning
* Provide financial insights through visualizations and reports

---

## 🚀 Key Features

* 👤 User login and authentication
* 📂 Upload and analyze bank statements
* 💸 Income and expense tracking
* 📊 Spending pattern analysis
* 🤖 Expense prediction using Machine Learning
* 🔔 Bill reminder notifications
* 🎯 Financial goal management
* 📑 Automatic PDF report generation

## ⚙️ Installation and Setup

### Step 1: Clone the Repository

```
git clone https://github.com/KIndhu06/SpendiQ.git
```

### Step 2: Navigate to Project Folder

```
cd SpendiQ
```

### Step 3: Install Required Libraries

```
pip install -r requirements.txt
```

### Step 4: Run the Application

```
streamlit run app.py
```

The application will start in your browser.
